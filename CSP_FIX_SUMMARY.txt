3. **Expected**: No CSP errors ✓

### Check CSP Header
1. DevTools → **Network** tab
2. Reload page
3. Click first request
4. Go to **Response Headers**
5. Find `Content-Security-Policy`
6. Should contain: `'nonce-xxxxxx'` (unique per request)
